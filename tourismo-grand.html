Tourismo Grand

  (2019)
Tourismo Grand is a platform that offers tour packages along with detailed descriptions and itineraries dynamically listed from the admin portal. There are different interfaces for vendors and travellers. Vendors can list the dynamically categorized tours from admin portal and users can see the active ones on website.

Current Scope :

• Auth : Login and register
• Create tours: Admins can create tours, associate them with respective categories and destinations.
• About us and contact page can be controlled by admin portal
• Users can see the listing of the categorized tours on the homepage.
• Users can like and share the tours on social networking platforms such as facebook, whatsapp.
• The views on each tour are recorded.
• Tour details page with corousels.
• Enquire about a product
• Raise a concern about a product.

Future Scope :


• Search and filter tours.
• Lifecycle of placing an order handled by the platform.
• Payment Gateway

Technology :

Codebase :
  Front-end : VueJS + vuex + vuetify + nuxt
  back-end : nodeJS + expressJS + JWT + MongoDB
